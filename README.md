# Next.js + TypeScript + Tailwind (App Router)

Simple project scaffolded in this folder. It includes a Tailwind-built card that demonstrates flexbox concepts.

Run locally:

```bash
npm install
npm run dev
# open http://localhost:3002

Nota: el servidor de desarrollo usa `webpack` por defecto (si quieres probar Turbopack, ejecuta `npm run dev:turbopack`).
```
